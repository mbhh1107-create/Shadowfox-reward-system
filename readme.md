# 🎁 ShadowFox Reward System

A web-based **Reward Management System** developed as part of the **ShadowFox Internship**. This application allows users to earn, track, and redeem reward points while providing administrators with tools to manage users, rewards, and transactions efficiently.

---

## 📌 Features

### 👤 User Features

* User registration and login
* Secure authentication
* View reward points balance
* Earn points through eligible activities or purchases
* Redeem available reward points
* View reward transaction history
* User-friendly dashboard

### 🛠️ Admin Features

* Manage user accounts
* Add, update, and delete rewards
* Monitor reward transactions
* Manage reward points
* View system statistics

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/ShadowFox-Reward-System.git
```

### 2. Navigate to the project folder

```bash
cd ShadowFox-Reward-System
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure MySQL

* Create a MySQL database.
* Import the provided SQL schema.
* Update your database credentials in the Python configuration file.

### 5. Run the application

```bash
python app.py
```

### 6. Open the application

Visit:

```
http://localhost:5000
```
